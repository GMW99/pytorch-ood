# Out-of-Distribution-Detection Toolkit

Python utilities to bootstrap research in fields related to Out-of-Distribution Detection, Open Set Recognition, 
Novelty and Anomaly Detection.

## Implemented Methods

- OpenMax
- ODIN Preprocessing
- Class Anchor Clustering
- II Loss
- CenterLoss 
- Triplet Loss
